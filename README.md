# Taskworld's RPC framework.

## Under construction.

## Development set-up

 1. Clone the repo:
    ```
    git clone git@github.com:taskworld/chayen.git
    cd chayen
    ```

 2. Install dependencies:
    ```
    yarn
    ```

 3. Spin up some Redis:
    ```
    docker-compose up -d
    ```

## Running tests

```
yarn test
```
